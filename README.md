# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project demonstrates core frontend development skills along with game logic implementation and DOM manipulation.

## 🚀 Features

Interactive and user-friendly UI

Two-player gameplay (Player X vs Player O)

Real-time game updates using JavaScript

Win detection and draw condition handling

Reset / Restart game functionality

Responsive design for different screen sizes

## 🛠️ Technologies Used

HTML – Structure of the game

CSS – Styling and layout

JavaScript – Game logic and DOM manipulation

## 🎯 How It Works

Players take turns clicking on the grid

The game tracks moves and updates the board dynamically

Automatically detects:

Winner (3 in a row)

Draw condition

Option to restart the game anytime
